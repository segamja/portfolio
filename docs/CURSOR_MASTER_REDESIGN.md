# Cursor 작업 지시

이 문서를 최우선으로 적용한다.

기존 영문 카피는 모두 폐기한다.

기존 번역 문구는 모두 폐기한다.

텍스트는 이 문서를 기준으로 다시 작성한다.

디자인은 Apple / Linear / Stripe / Vercel 수준의 Premium SaaS Landing Page 느낌으로 리디자인한다.

코드 품질과 기능은 유지한다.

빌드가 성공할 때까지 수정한다.
